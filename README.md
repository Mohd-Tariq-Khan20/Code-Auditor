# Socratic-AI-Code-Auditor
An advanced AI-powered static analysis platform and production-ready workspace engineered to audit, optimize, and evaluate source code quality across multiple programming paradigms.

## 🚀 Key Modules & Architecture

* **🔍 Core Code Auditor:** Automates security vulnerability scanning (such as buffer overflows or injection risks) with comprehensive risk indexing.
* **🏆 Competitive Analysis Engine:** Evaluates execution complexities ($O(N^2)$ to $O(N)$ updates) and loops against strict problem runtime constraints to catch Time Limit Exceeded (TLE) vulnerabilities.
* **🎯 Interactive Coding Arena:** Implements a dynamic sandbox environment with active secure patching models for daily language-specific challenges.
* **🧠 Socratic Guidance Architecture:** Driven by custom LLM prompt design (Meta Llama-3.3-70B via NVIDIA API) that strategically provides targeted performance metrics and architectural hints without direct code leaks.
* **⚙️ Smart Execution Workspace:** Integrated with localized browser injection wrappers for compiler-style smart brackets matching, automated inline spellcheck bypass, and persistent historical transaction logging powered by SQLite.

## 🛠️ Tech Stack
* **Frontend/Dashboard:** Streamlit Engine
* **Core Language:** Python
* **Database Layer:** SQLite3 with Custom State Synchronization
* **AI Orchestration:** OpenAI API Wrapper (NVIDIA NIM Endpoint Cluster)
